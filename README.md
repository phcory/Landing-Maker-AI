\# 🌟 Landing Maker IA Tool 🌟

The Landing Maker IA tool is a web application designed to generate professional-looking webpages using artificial intelligence. This project aims to provide a user-friendly interface for users to input a prompt and receive a fully functional webpage that meets their needs. The tool integrates with various technologies, including Google Fonts, CSS styling, and JavaScript, to provide a seamless user experience.



\## 🚀 Features

\- User input prompt to describe their business

\- Generation of a professional-looking webpage based on the user's input

\- Display of the generated webpage within an iframe element

\- Display of the generated code within a p element

\- Integration with Google Fonts for styling

\- Use of CSS styling for additional layout and design

\- JavaScript code to handle user interactions and generate the webpage



\## 🛠️ Tech Stack

\* Frontend: HTML, CSS, JavaScript

\* Styling: Google Fonts, CSS

\* Scripting: JavaScript

\* Dependencies: Google Fonts, style.css, scripts.js

\* Build Tools: None

\* AI Tools: Artificial intelligence for webpage generation
